# Zadanie rekrutacyjne

### Proces zbudowania i uruchomienia aplikacji

- Otworzyć projekt w terminalu
- `yarn`
- `yarn start`
- Otworzyć kolejny terminal projektu i użyć komendy `react-native run-android`
